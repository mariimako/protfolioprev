JSLearn

Learn the Japan Sign Language using Tensorflow Object Detection Models

This project was created when I realized how it may be difficult for communication among sign language users across different contries because of their different sign language alphabets. By implementing an object detection model customized for the JSL (Japan Sign Language), I enabled users to interactively learn the Japan Sign Language!

Users are able to start a webcam from the website that starts the usage of the object detection model. Than, every frame, the video is captured and fed into a pretrained model that compares it to different hand signatures.

Each set of characters (such as あ〜お）where separated to different models to make the website run smoother and the detection process to be smoother. I was able to not only learn introductory machine learning (I would mention, it did not involve much complex math, as I did not create a neural network from scratch, but rather building on existing frameworks) with Python and Tensorflow, but also implementing it in web applications in a user friendly way.
